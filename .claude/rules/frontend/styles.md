# Design System

## Colors
- Sidebar background: #0B1F3A (navy)
- Card background: white (#FFFFFF)
- Accent / CTA: #E8451A (orange-red)
- Text primary: #111827
- Text muted: #6B7280

## Status badge colors
- On Track: green (bg-green-100 text-green-700)
- At Risk: amber (bg-amber-100 text-amber-700)
- Blocked: red (bg-red-100 text-red-700)
- Slipped: orange-red (bg-orange-100 text-orange-700)
- In Progress: blue (bg-blue-100 text-blue-700)

## shadcn/ui components in use
- Button, Card, CardHeader, CardContent, Badge, Tabs, TabsList, TabsTrigger, TabsContent
- Avatar, AvatarFallback, Separator, Progress
- Always import from @/components/ui/<name>

## Layout
- Left sidebar: fixed width 240px, navy background
- Main content: white, full remaining width
- Page padding: p-6
- Card grid: 2 columns on dashboard (grid-cols-2 gap-4)

## Typography
- Page titles: text-2xl font-semibold
- Section headers: text-lg font-medium
- Body: text-sm text-gray-700
- Muted labels: text-xs text-gray-500 uppercase tracking-wide
